# basketball-bund-api
Dokumentation der öffentlich nachverfolgbaren REST-API-Endpunkte von Basketball-bund.net basierend auf legitimen Browser-Entwicklertools. Enthält keine proprietären Daten; dient ausschließlich zu Bildungs- und Dokumentationszwecken unter Berücksichtigung aller geltenden rechtlichen Vorgaben.
